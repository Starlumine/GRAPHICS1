Garden Guardian Documentation

Name: Garden Guardian

Description

A game where the player protects nine flower plots from aphids while keeping them hydrated. The Arduino reads joystick movements and button presses, controls LEDs representing each flower’s health, and communicates via WebSerial to a p5.js front-end that renders the scene and plays sound effects with Tone.js.

Gameplay Loop

  1. Start Screen: Displays title and “Press Button to Start.”
  2. Gameplay: Water flower plots with the Arduino button to keep them healthy while attacking aphids with the button as well. When watering plants there is an animation to show that it is successfully watered. Flower plots have three stages. When it gets to its wilting stage due to not watering it, the flowers will die in a couple seconds. When aphids land on a targeted flower, it takes a couple seconds for the flower to be destroyed.
  3. End Screen: After time expires or the flowers are all destroyed, the end screen displays final stats and high score.

Arduino (C++)

  • Inputs:
    • Joystick X/Y → analog pins A0/A1 (player movement)
    • Button → digital pin 3 (water/repel action)
  • Outputs:
    • LEDs on pins 4–12 for each flower’s alive/dead state
    • Tone.js Audio: Plays background loop, watering, and attacking sounds

Future Development

For future development, difficulty should be changed so that the experience is easier for people to get into playing. Also should add an instructions page, backgrounds, and new enemy types.
